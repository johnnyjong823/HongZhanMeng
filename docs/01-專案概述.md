# 鴻展盟專案 - PHP MVC 實作文件

## 專案概述

### 專案資訊
- **專案名稱**: 鴻展盟管理系統
- **開發語言**: PHP 8.x
- **資料庫**: MySQL 5.7+ / MariaDB 10.x+
- **部署環境**: cPanel + FTP
- **架構模式**: MVC (Model-View-Controller)

### 主機需求

#### cPanel 主機最低需求
| 項目 | 最低需求 | 建議需求 |
|------|----------|----------|
| PHP 版本 | 8.0+ | 8.2+ |
| MySQL 版本 | 5.7+ | 8.0+ |
| 記憶體限制 | 128MB | 256MB |
| 上傳檔案限制 | 32MB | 64MB |
| 執行時間限制 | 30秒 | 120秒 |

#### 必要 PHP 擴充套件
```
- pdo_mysql (MySQL 資料庫連接)
- mbstring (多位元組字串處理)
- openssl (加密功能)
- json (JSON 處理)
- session (Session 管理)
- fileinfo (檔案資訊)
- gd 或 imagick (圖片處理，選用)
```

#### cPanel 設定確認事項
1. **PHP 版本設定**: 在 cPanel > Select PHP Version 確認 PHP 8.0+
2. **MySQL 資料庫**: 在 cPanel > MySQL Databases 建立資料庫
3. **檔案權限**: 確保 `storage/` 和 `cache/` 資料夾權限為 755 或 775

## 系統功能概述

### 核心功能
1. **帳號管理系統** (AcUsers)
2. **權限管理系統** (AcRoles)
3. **功能畫面管理** (AcFunction)
4. **操作紀錄系統** (ActionLogs)
5. **登入紀錄系統** (LoginLogs)

### 前後台分離
- **前台**: 公開頁面，不需登入，網址以 `/` 開頭
- **後台**: 管理頁面，需要登入，網址以 `/admin/` 開頭
- **登入頁面**: `/account/login`

### 權限層級
| 層級 | 角色代碼 | 角色名稱 | 說明 |
|------|----------|----------|------|
| 1 | admin | 最大管理者 | 開發者使用，可管理所有功能 |
| 2 | host | 使用者管理者 | 客戶最大管理權限 |
| 3 | user | 使用者 | 客戶一般管理者 |

### 權限授權規則
- **最大管理者 (admin)**: 可授權給「使用者管理者 (host)」
- **使用者管理者 (host)**: 可授權給「使用者 (user)」自己擁有的權限
- **使用者 (user)**: 無法授權給其他人

### Session 管理
- 登入後 60 分鐘無操作自動登出
- 每次操作自動更新 Session 時間
